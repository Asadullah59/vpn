Features

Cloud Network

Create a self-hosted secure cloud network for LAN-like connectivity between remote users.

Easy Configuration

All user and server configuration is done from a simple web interface.

User Management

Easily manage multiple organizations, users and vpn servers.

Security

All traffic between your computer and the vpn server is encrypted.

Anonymity

Improve your online anonymity, all your internet traffic will be routed through your vpn server.

Two-Step Authentication

Optional two-step authentication is available using Google Authenticator for improved security.

REST API

Easily intergrate your cloud network with other services using the REST API.

Frequently Asked Questions

What VPN modes are available?

Three modes are available to route all internet traffic, route only specified subnets or route only VPN traffic.

How can bugs be reported?

Bugs can be reported on the GitHub Issues page

What does enterprise subscription include?

The $2.50 enterprise subscription includes server bandwidth graphs, user searching, detailed user information, static user IP addresses and temporary key links.

What is the default username and password?

The default username and password is "admin".

Features

Cloud Network

Create a self-hosted secure cloud network for LAN-like connectivity between remote users.

Easy Configuration

All user and server configuration is done from a simple web interface.

User Management

Easily manage multiple organizations, users and vpn servers.

Security

All traffic between your computer and the pritunl server is encrypted.

Anonymity

Improve your online anonymity, all your internet traffic will be routed through your vpn server.

Two-Step Authentication

Optional two-step authentication is available using Google Authenticator for improved security.

REST API

Easily integrate and configure Pritunl with other services using the REST API.

Use Cases

Build a cloud network in minutes

Easily create a self-hosted cloud network to allow for vpn networking between users to securely share resources.

Virtualize your office

Provide easy and secure remote access to the local office resources to remote users.

Remote access to servers

Create a gateway to your AWS VPC or VPS private network to securly access your servers.

Improve security and privacy

Protect your internet traffic when connected to WiFi or while on an untrusted network.

Frequently Asked Questions

What VPN modes are available?

Three modes are available to route all internet traffic, route only traffic to specified subnets or route only VPN traffic.

How can bugs be reported?

Bugs can be reported on the GitHub Issues page or with the contact widget below.

What does enterprise subscription include?

The $2.5 subscription includes bandwidth graphs, user searching, detailed user information, static user IP's and Postmark integration for emailing temporary key links.

What is the default username and password?

The default username and password is "admin".

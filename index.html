pritunl
simple self-hosted openvpn server
DEMO INSTALL

Features

  • Simple web interface

    All user and server configuration is done from a simple web interface. Protected by a username and password login.

  • Server management

    Multiple vpn servers can be created and managed on one server within the web interface.

  • User management

    Manage multiple organizations and users. Users certificates are automatically disabled when user is deleted.

  • Anonymity

    Improve your online anonymity, all your internet traffic will appear to be coming from your vpn server this hides your public IP and the geolocation of your IP.

  • Security

    All traffic between your computer and the vpn server is encrypted. The traffic can't be monitored or exploited by your ISP/college or while on public wifi.

  • Bypass Censors

    Bypass censored websites and blocked ports. Websites that block access to countries outside the US can also be accessed if your VPN server is located in the US.
